## Когда использовать

Условные предложения отвечают на вопрос «что будет/что бывает, ЕСЛИ…». Есть две части: условие с **if** и результат.

**Zero conditional (нулевое)** — про то, что **всегда** правда: законы природы, факты, привычки, инструкции. Здесь *if* почти равно *when* («когда / всякий раз когда»).

- If you heat water to 100°C, it boils. — факт, так всегда.
- If I drink coffee at night, I don't sleep. — моя привычка.

**First conditional (первое)** — про **реальное** будущее: одно конкретное событие, которое вполне может произойти.

- If it rains tomorrow, we will stay home. — дождь реально возможен.
- If you study hard, you will pass the exam. — реальный шанс.

> **Главное правило (запомни как формулу):** после **if** мы НЕ ставим **will**. Будущее в части с *if* показываем через **present simple**.

## Форма

Запятая нужна, только если предложение **начинается** с *if*. Если *if* в середине — запятой нет.

**Zero conditional — оба глагола в present simple:**

| If-часть (условие) | Главная часть (результат) |
|--------------------|---------------------------|
| If + present simple | present simple |
| If you **press** this button, | the machine **stops**. |
| Ice **melts** | if you **heat** it. |

**First conditional — if + present simple, will в результате:**

| If-часть (условие) | Главная часть (результат) |
|--------------------|---------------------------|
| If + present simple | will + глагол (база) |
| If it **rains**, | we **will stay** home. |
| I **will call** you | if I **am** late. |

**Сравни две модели:**

| Тип | If-часть | Результат | Смысл |
|-----|----------|-----------|-------|
| Zero | If you **mix** red and blue, | you **get** purple. | всегда так |
| First | If you **mix** these two, | it **will explode**. | один конкретный раз |

⚠️ В обеих моделях глагол после *if* стоит в **present simple** (для he/she/it — с **-s**: if she *comes*). Will появляется только в части БЕЗ *if*.

## Частые ошибки
- ❌ If it **will rain**, we will stay home → ✅ If it **rains**, we will stay home *(после if — present simple, без will)*.
- ❌ If you **will heat** ice, it melts → ✅ If you **heat** ice, it melts *(zero conditional: will нет вообще)*.
- ❌ If she comes, I **tell** her → ✅ If she comes, I **will tell** her *(в first conditional результат с will)*.
- ❌ If she **come** tomorrow → ✅ If she **comes** tomorrow *(present simple, he/she/it → -s)*.

## Примеры
- If you heat ice, it melts. — Если нагреть лёд, он тает. *(факт — zero)*
- If I'm tired, I go to bed early. — Если я устаю, я ложусь спать рано. *(привычка — zero)*
- If it rains tomorrow, we will stay at home. — Если завтра пойдёт дождь, мы останемся дома. *(будущее — first)*
- If you study hard, you will pass the test. — Если ты будешь усердно учиться, ты сдашь тест. *(first)*
- I will call you if I am late. — Я позвоню тебе, если буду опаздывать. *(if в середине — запятой нет)*
- If she doesn't hurry, she will miss the train. — Если она не поторопится, она опоздает на поезд. *(first, отрицание)*
