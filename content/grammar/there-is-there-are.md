## Когда использовать
Оборот **there is / there are** = "есть, имеется, находится". Так мы сообщаем, что что-то **существует** или **где-то находится**.

- **There is** a problem. — Есть проблема.
- **There are** two windows in the kitchen. — На кухне два окна.

Главная идея: глагол (is/are) согласуется с тем, что идёт **после** него (с реальным предметом), а не со словом *there*.

> Слово **there** здесь — НЕ "там". Это формальное начало фразы. "Там" в значении места — это тоже *there*, но в конце: *There is a cat **there**.* (Там есть кошка.)

## Форма
Выбор is/are зависит от числа предмета:

| Число | Оборот | Пример |
|-------|--------|--------|
| один (singular) | **there is** (there's) | There is **a** car. |
| много (plural) | **there are** | There are **three** cars. |

**Отрицание** (+ not / no):
| + | − |
|---|---|
| There is a chair. | There **is not** (isn't) a chair. / There is **no** chair. |
| There are chairs. | There **are not** (aren't) any chairs. / There are **no** chairs. |

> Два способа отрицания: **isn't/aren't + any** ИЛИ **is/are + no**. Нельзя смешивать: ❌ *isn't no*.

**Вопрос** — is/are выходят вперёд:
| Утверждение | Вопрос |
|-------------|--------|
| There **is** a bank here. | **Is there** a bank here? |
| There **are** shops here. | **Are there** any shops here? |

Краткие ответы: Yes, there is. / No, there isn't. — Yes, there are. / No, there aren't.

## a / some / any
- **Один** предмет → **a/an**: There is **a** dog.
- **Много** (точное число) → число: There are **five** dogs.
- **Много** (неопределённо) в утверждении → **some**: There are **some** dogs.
- В вопросе и отрицании → **any**: Are there **any** dogs? / There aren't **any** dogs.

## Частые ошибки
- ❌ There is two beds. → ✅ There **are** two beds. *(много → are)*
- ❌ There are a book. → ✅ There **is** a book. *(один → is)*
- ❌ In the room is a table. → ✅ **There is** a table in the room. *(нужно слово there)*
- ❌ There is not any chairs. → ✅ There **are not** any chairs. *(chairs мн. → are)*

## Примеры
- **There is** a cat under the table. — Под столом есть кошка.
- **There are** many people here. — Здесь много людей.
- **Is there** a hotel near the station? — Рядом с вокзалом есть отель?
- **There isn't** any milk in the fridge. — В холодильнике нет молока.
- **There are no** free seats. — Свободных мест нет.
- How many students **are there** in your class? — Сколько студентов в твоём классе?
