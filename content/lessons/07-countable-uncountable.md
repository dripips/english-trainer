## Часть 1. Два типа существительных

**Исчисляемые (Countable)** — можно посчитать:
- a book / **two** books — книга / две книги
- a chair / **three** chairs — стул / три стула
- an idea / **five** ideas — идея / пять идей

**Неисчисляемые (Uncountable)** — нельзя посчитать:
- **water** (not a water / two waters) — вода
- **money** (not a money) — деньги
- **information** (not an information) — информация

> 💡 Неисчисляемые в русском тоже есть: «вода», «деньги», «информация». Проблема в том, что некоторые слова неисчисляемые только в английском.

## Часть 2. Частые неисчисляемые (запомни список)

| Слово | Ловушка |
|---|---|
| **information** | ❌ informations, ❌ an information |
| **advice** | ❌ advices, ❌ an advice |
| **news** | ❌ a news (The news is good.) |
| **money** | ❌ moneys |
| **furniture** | ❌ furnitures (I need some furniture.) |
| **luggage / baggage** | ❌ luggages |
| **work** | ❌ a work (= произведение). Обычно: I have a lot of **work**. |
| **traffic** | ❌ traffics |
| **weather** | ❌ a weather |
| **progress** | ❌ progresses |
| **bread** | ❌ a bread (но: **a loaf of** bread, **a slice of** bread) |
| **hair** | ❌ hairs (обычно, в общем смысле) |

## Часть 3. Как сказать «немного/много» — таблица

| | Исчисляемые | Неисчисляемые |
|---|---|---|
| **Немного (+)** | **a few** friends | **a little** water |
| **Мало (−, нехватка)** | **few** friends | **little** water |
| **Много** | **many** books | **much** time |
| **Много (нейтрально)** | **a lot of** books | **a lot of** time |
| **В вопросах** | **How many** books? | **How much** water? |

> 💡 **a lot of** — самое универсальное, работает с обоими типами. В разговорной речи используй его вместо many/much.

---

## Часть 4. Some vs Any

| | Утверждение | Вопрос | Отрицание |
|---|---|---|---|
| **Исчисляемые (мн.ч.)** | I have **some** books. | Do you have **any** books? | I don't have **any** books. |
| **Неисчисляемые** | There is **some** water. | Is there **any** water? | There isn't **any** water. |

Исключение: **some** в вопросах-предложениях:
- Would you like **some** tea? — Хочешь чай? (предлагаем)
- Can I have **some** water? — Можно воды? (просьба)

---

## Часть 5. «Порция» неисчисляемого

Чтобы посчитать неисчисляемое — добавь «контейнер»:

- **a cup of** tea/coffee — чашка чая/кофе
- **a glass of** water/juice — стакан воды/сока
- **a bottle of** water — бутылка воды
- **a slice of** bread/cake — кусок хлеба/торта
- **a piece of** advice/furniture/information — совет/мебель/информация
- **a loaf of** bread — буханка хлеба
