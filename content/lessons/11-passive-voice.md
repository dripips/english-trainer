## Часть 1. Зачем нужен Passive Voice

**Активный залог** — говорим кто сделал:
- Tolstoy **wrote** War and Peace. — Толстой написал «Войну и мир».

**Страдательный залог** — говорим что произошло с объектом:
- War and Peace **was written** by Tolstoy. — «Война и мир» была написана Толстым.

Когда использовать Passive:
1. **Исполнитель неизвестен:** My phone **was stolen**.
2. **Исполнитель неважен:** English **is taught** in schools.
3. **Официальный/научный стиль:** The results **were recorded**.
4. **Хочем подчеркнуть объект:** Three people **were injured**.

---

## Часть 2. Формула: be + V3

| Время | Формула | Пример |
|---|---|---|
| Present Simple | is / are + V3 | English **is spoken** here. |
| Past Simple | was / were + V3 | The letter **was sent** yesterday. |
| Future Simple | will be + V3 | It **will be done** tomorrow. |
| Present Continuous | is/are being + V3 | The road **is being repaired**. |
| Present Perfect | has/have been + V3 | She **has been promoted**. |
| Past Perfect | had been + V3 | It **had been built** before 1900. |
| Modal | modal + be + V3 | It **must be signed**. |

---

## Часть 3. By — кем сделано

Если важно назвать исполнителя — добавляем **by**:
- The film **was directed** **by** Nolan. — Фильм был снят Ноланом.
- I **was surprised** **by** the news. — Я был удивлён новостью.

Если исполнитель очевиден или неважен — **by** опускаем:
- The window **was broken**. *(кем — не важно)*

---

## Часть 4. Активный → Пассивный

| Активный | Пассивный |
|---|---|
| Shakespeare **wrote** Hamlet. | Hamlet **was written** by Shakespeare. |
| They **are building** a new school. | A new school **is being built**. |
| Someone **has stolen** my bike. | My bike **has been stolen**. |
| They **will announce** the results. | The results **will be announced**. |

**Правило перевода:**
1. Объект активного → подлежащее пассивного
2. Глагол → be (в нужном времени) + V3
3. Подлежащее активного → by + … (если нужно)

---

## Часть 5. Неправильные V3 (повторение)

| Глагол | V3 |
|---|---|
| build | **built** |
| write | **written** |
| make | **made** |
| take | **taken** |
| give | **given** |
| send | **sent** |
| find | **found** |
| break | **broken** |
| steal | **stolen** |
| tell | **told** |
| know | **known** |
| hold | **held** |
