## Часть 1. Что такое First Conditional

First Conditional используем для **реальных и вероятных** ситуаций в будущем.

**Формула:**
> **If** + Present Simple, **will** + инфинитив

Примеры:
- **If** it **rains**, I **will** take an umbrella. — Если будет дождь, я возьму зонт.
- **If** you **study** hard, you **will** pass. — Если будешь усердно заниматься, сдашь.

---

## Часть 2. Главная ловушка

🔴 **В части с if — никакого will!**

| ❌ Неверно | ✅ Верно |
|---|---|
| If it **will rain**… | If it **rains**… |
| If she **will come**… | If she **comes**… |
| If you **will try**… | If you **try**… |

В русском мы говорим «если **будет** дождь» — с будущим временем. В английском if-часть всегда Present Simple!

---

## Часть 3. Порядок частей

Порядок можно менять — смысл не меняется:

- **If it rains**, I'll stay home.
- I'll stay home **if it rains**.

> 💡 Если if-часть стоит первой — ставим запятую. Если второй — запятая не нужна.

---

## Часть 4. Отрицание

В if-части: **if + don't / doesn't**
- If you **don't hurry**, you'll be late.
- If she **doesn't call**, I'll be worried.

В главной части: **won't** (= will not)
- If it rains, we **won't** go out.
- She **won't** pass if she doesn't study.

---

## Часть 5. Zero vs First Conditional

| | Zero Conditional | First Conditional |
|---|---|---|
| **Когда** | Всегда правда (факты, законы) | Реальное будущее |
| **Формула** | If + Present Simple, Present Simple | If + Present Simple, will + V |
| **Пример** | If you heat water to 100°C, it **boils**. | If it rains tomorrow, I **will** stay home. |

---

## Часть 6. Полезные фразы с First Conditional

- **Unless** = if … not:
  - **Unless** you study, you **won't** pass. = If you **don't** study, you **won't** pass.

- **As long as / provided that** = при условии что:
  - I'll help you **as long as** you try.

- **When** вместо **if** (когда уверены что произойдёт):
  - **When** I finish work, I'll call you. (точно закончу)
  - **If** I finish work early, I'll call you. (возможно закончу рано)
