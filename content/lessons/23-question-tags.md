## Часть 1. Правило

Question tag добавляется в конце предложения для подтверждения или уточнения.

**Главное правило:**
- Утвердительное предложение → **отрицательный** тег
- Отрицательное предложение → **положительный** тег

| Предложение | Тег |
|---|---|
| It **is** cold, | **isn't** it? |
| She **can** swim, | **can't** she? |
| They **were** late, | **weren't** they? |
| He **doesn't** like it, | **does** he? |
| You **haven't** eaten, | **have** you? |
| We **won't** go, | **will** we? |

---

## Часть 2. Какой вспомогательный глагол использовать

| Основное предложение | Тег |
|---|---|
| is / am / are | isn't / aren't |
| was / were | wasn't / weren't |
| do / does | don't / doesn't |
| did | didn't |
| have / has | haven't / hasn't |
| will / won't | won't / will |
| can / can't | can't / can |
| should | shouldn't |
| would | wouldn't |

Если вспомогательного нет → используй **do/does/did**:
- She **works** here, **doesn't** she?
- They **left** early, **didn't** they?

---

## Часть 3. Исключения

| Предложение | Тег |
|---|---|
| **I am** right, | **aren't I**? *(не amn't I)* |
| **Let's** go, | **shall we**? |
| **Don't** move, | **will you**? |
| **Nobody** called, | **did they**? *(nobody → they)* |
| **Everyone** agrees, | **don't they**? |
| **There is** a problem, | **isn't there**? |

---

## Часть 4. Интонация

Интонация меняет смысл:

**Понижающаяся** (falling) → ты уверен, просто подтверждаешь:
- It's beautiful, **isn't it**? ↘ *(Красиво, правда?)*

**Повышающаяся** (rising) → ты не уверен, спрашиваешь:
- You've met before, **haven't you**? ↗ *(Вы знакомы?)*
