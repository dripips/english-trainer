## Часть 1. Зачем нужно словообразование

В IELTS Reading, Writing и Use of English постоянно встречаются задания где нужно:
- Поставить правильную форму слова в контексте
- Понять незнакомое слово по корню + суффиксу/префиксу

**Зная корень + правила**, можно понять и построить сотни слов.

---

## Часть 2. Суффиксы существительных

| Суффикс | Пример | Откуда |
|---|---|---|
| **-tion / -ation** | education, information, communication | educate, inform, communicate |
| **-ment** | development, government, achievement | develop, govern, achieve |
| **-ness** | happiness, kindness, darkness | happy, kind, dark |
| **-ity / -ty** | ability, creativity, safety | able, creative, safe |
| **-ance / -ence** | performance, confidence, independence | perform, confident, independent |
| **-er / -or** | teacher, director, actor | teach, direct, act |
| **-ist** | scientist, journalist, specialist | science, journal, special |
| **-ism** | capitalism, criticism, tourism | capital, critic, tour |
| **-ship** | friendship, leadership, relationship | friend, lead, relate |
| **-hood** | childhood, neighbourhood | child, neighbour |

## Часть 3. Суффиксы прилагательных

| Суффикс | Пример | Откуда |
|---|---|---|
| **-ful** | careful, successful, powerful | care, success, power |
| **-less** | careless, hopeless, harmless | care, hope, harm |
| **-able / -ible** | comfortable, possible, responsible | comfort, —, — |
| **-ive** | creative, effective, productive | create, effect, produce |
| **-al** | educational, natural, cultural | education, nature, culture |
| **-ous** | famous, dangerous, ambitious | fame, danger, ambition |
| **-ical** | political, musical, historical | politics, music, history |
| **-y** | healthy, rainy, sunny | health, rain, sun |
| **-ic** | economic, scientific, dramatic | economy, science, drama |

---

## Часть 4. Суффиксы наречий

Большинство наречий: **прилагательное + -ly**:
- quick → **quickly**
- careful → **carefully**
- beautiful → **beautifully**
- serious → **seriously**

Исключения:
- good → **well** *(не goodly)*
- fast → **fast** *(не fastly)*
- hard → **hard** *(not hardly — у него другое значение: едва)*
- late → **late** *(not lately — у него другое значение: в последнее время)*

| Суффикс | Пример | Откуда |
|---|---|---|

| **-en** | strengthen, widen, shorten | strong, wide, short |

## Часть 6. Префиксы

### Отрицание
| Префикс | Пример |
|---|---|
| **un-** | unhappy, unclear, unusual |
| **in-** | incorrect, incomplete, indirect |
| **im-** (перед p/m/b) | impossible, impolite, immature |
| **ir-** (перед r) | irresponsible, irregular, irrational |
| **il-** (перед l) | illegal, illogical |
| **dis-** | disagree, disappear, dishonest |
| **mis-** | misunderstand, mistake, misuse |
| **non-** | non-fiction, non-stop, non-profit |

### Другие значения
| Префикс | Значение | Пример |
|---|---|---|
| **re-** | снова | rebuild, redo, reconsider |
| **pre-** | до | prepare, predict, prevent |
| **over-** | слишком/через | overwork, overuse, overcome |
| **under-** | мало/под | underestimate, underline |
| **co-** | вместе | cooperate, co-author |
| **inter-** | между | international, interact |
| **multi-** | много | multicultural, multinational |

## IELTS Word Formation — типичное задание

```
The ___ (develop) of technology has changed our lives.
→ development

She showed great ___ (determine).
→ determination

He acted very ___ (responsible).
→ irresponsibly
```

**Алгоритм:**
1. Определи что нужно по контексту (noun/adj/adv/verb)
2. Найди корень
3. Применяй суффикс + префикс если нужно
