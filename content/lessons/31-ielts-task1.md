## Структура IELTS Task 1 (150+ слов, 20 мин)

```
§1 ВВЕДЕНИЕ (1-2 предл.)
   Перефразируй задание. НЕ копируй слово в слово.

§2 ОБЗОР — OVERVIEW (2-3 предл.)
   Главные тенденции. Начни с "Overall, ..."
   Это самый важный абзац!

§3 ДЕТАЛИ А (3-4 предл.)
   Конкретные цифры для первой группы данных.

§4 ДЕТАЛИ Б (3-4 предл.)
   Конкретные цифры для второй группы данных.
```

> **Никогда не пиши своё мнение в Task 1!**
> ❌ I think this is interesting.
> ✅ Overall, the data suggests an upward trend.

## Словарный банк

### Рост (Increase)
| Глагол | Существительное | Наречие |
|---|---|---|
| increase / rise / grow | an increase / a rise / growth | sharply |
| climb / go up / surge | a surge / a jump | dramatically |
| double / triple | — | significantly |
| peak at (достичь пика) | a peak | steadily |

### Падение (Decrease)
| Глагол | Существительное | Наречие |
|---|---|---|
| decrease / fall / drop | a decrease / a fall / a drop | sharply |
| decline / go down / plunge | a decline / a plunge | dramatically |
| halve (уменьшиться вдвое) | — | gradually |
| bottom out (достичь дна) | a low | slightly |

### Стабильность
- remain stable / stay constant / level off
- there was little change in…
- figures stayed at approximately…

### Сравнение
- compared to / in comparison with
- while / whereas
- … was higher than … / … accounted for the largest share

## Стандартные фразы для введения

| Тип графика | Введение |
|---|---|
| Line graph | The line graph illustrates… |
| Bar chart | The bar chart compares… |
| Pie chart | The pie chart shows the proportion of… |
| Table | The table provides data on… |
| Map | The maps show changes to… |
| Process | The diagram illustrates the process of… |

---

## Полезные конструкции

**Приблизительные цифры:**
- approximately / roughly / around / about 50%
- just over / just under 50%
- nearly / almost 50%

**Сравнение:**
- X was twice as high as Y
- X accounted for 30%, compared to 20% for Y
- The proportion of X was significantly higher than that of Y

**Указание времени:**
- between 2000 and 2020 / from 2000 to 2020
- over the period / throughout the decade
- by 2020 (к 2020 году) / in 2020 (в 2020 году)

## Шаблон Overview

```
Overall, it is clear that [главная тенденция].
While [одна группа] showed [тренд], [другая группа] [контрастный тренд].
[Самое заметное наблюдение].
```

Пример:
> Overall, it is clear that all three countries experienced growth in tourism. While France maintained the highest figures throughout, Italy showed the most dramatic increase over the period.
