## Топ-50 слов Academic Word List

### Глаголы
| Слово | Значение | Коллокация |
|---|---|---|
| **analyse** | анализировать | analyse data/results |
| **assess** | оценивать | assess the impact/risk |
| **assume** | предполагать | assume that / on the assumption that |
| **demonstrate** | демонстрировать | demonstrate the ability/importance |
| **derive** | выводить, извлекать | derive conclusions from |
| **establish** | устанавливать | establish a link/context |
| **evaluate** | оценивать (качество) | evaluate the evidence |
| **identify** | определять | identify the key issues |
| **indicate** | свидетельствовать | indicate a trend / the data indicates |
| **involve** | включать, требовать | involve doing / involve several steps |
| **obtain** | получать | obtain results/a degree |
| **occur** | происходить | occur frequently / when X occurs |
| **require** | требовать | require further research |
| **suggest** | свидетельствовать о | the evidence suggests |
