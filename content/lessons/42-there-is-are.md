## Часть 1. Зачем нужны there is / there are

Когда мы хотим сказать, что что-то **где-то есть** (существует, находится), мы используем **there is** или **there are**. Это как русское «есть» / «находится» / «имеется».

- **There is** a cat in the garden. — В саду есть кошка.
- **There are** two cats in the garden. — В саду есть две кошки.

> ⚠️ Слово **there** здесь НЕ переводится как «там». Это просто начало фразы. Перевод начинаем с конца: «В саду... есть кошка».

---

## Часть 2. There is или There are?

Выбор зависит от того, **сколько** предметов: один или много.

| Сколько | Форма | Пример |
|---|---|---|
| Один предмет (единственное число) | **There is** | There is **a** book. |
| Много предметов (множественное число) | **There are** | There are **two** books. |
| Неисчисляемое (milk, water, bread) | **There is** | There is some milk. |

Короткая форма: **There's** a book. (there is = there's)

---

## Часть 3. Отрицание (чего-то НЕТ)

Просто добавляем **not**:

- There **isn't** a TV. — Телевизора нет. *(is not = isn't)*
- There **aren't** any chairs. — Стульев нет. *(are not = aren't)*

| Число | Отрицание | Пример |
|---|---|---|
| Единственное | there **isn't** | There isn't a lamp. |
| Множественное | there **aren't** | There aren't any people. |

---

## Часть 4. Вопросы (есть ли?)

Выносим **is** или **are** вперёд:

- **Is there** a bank near here? — Есть ли рядом банк?
- **Are there** any shops? — Есть ли магазины?

**Короткие ответы:**

| Вопрос | Да | Нет |
|---|---|---|
| Is there...? | Yes, there **is**. | No, there **isn't**. |
| Are there...? | Yes, there **are**. | No, there **aren't**. |

---

## Часть 5. some и any

- **some** — «несколько / немного» в **утверждениях**: There are **some** apples.
- **any** — в **отрицаниях** и **вопросах**: There aren't **any** apples. · Are there **any** apples?

| Тип фразы | Слово | Пример |
|---|---|---|
| Утверждение | **some** | There are some books. |
| Отрицание | **any** | There aren't any books. |
| Вопрос | **any** | Are there any books? |

---

## Часть 6. Частые ошибки

1. **Число не совпадает:**
   - ❌ There **are** a book. → ✅ There **is** a book. *(одна книга)*

2. **Забыли про множественное число:**
   - ❌ There **is** many people. → ✅ There **are** many people.

3. **Неправильное слово в вопросе:**
   - ❌ **Is** there any shops? → ✅ **Are** there any shops? *(shops — много)*
